# timezone_web_simple


### Steps to run/open webpage

#### Ensure Flask is installed with Python
```bash
  $ pip3 install flask
```

#### Method 1:
```bash
  $ cd /(location of timezone_web.py)
  $ python3 timezone.py
```

#### Method 2:
```bash
  $ cd /(location of timezone_web.py)
  $ export FLASK_APP=timezone_web
  $ flask run
```

#### If succesful, open browser to http://127.0.0.1:500/ to view the webpage
